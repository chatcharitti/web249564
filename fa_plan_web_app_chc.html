🚀 จุดเริ่มต้นของการพัฒนา

Web App : FA Plan Tool v.1 ที่ท่านได้ลองใช้งานอยู่ชุดนี้ พัฒนาขึ้นด้วยเทคโนโลยี AI จาก claude.ai และ Gemini 3 จึงสามารถพัฒนาได้อย่างรวดเร็วในเวลาอันสั้น

📊 จาก Excel สู่ Web Application

เดิมทีผู้พัฒนาใช้วิธีคำนวณด้วย MS Excel ซึ่งเรียนรู้มาจากการฝึกอบรมต่างๆ ทั้งหลักสูตร IARFC แบบจ่ายเงินอบรมเอง และเรียนกับอาจารย์ที่เป็น CFP เมื่อครั้งเป็น รอง ผอ. ฝ่ายฝึกอบรมของบริษัทประกันอันดับต้นๆ ของไทย จากนั้นจึงนำความรู้และต้นแบบ Excel มาต่อยอดและปรับปรุง เพิ่มเติมสิ่งต่างๆ เพื่อใช้นำเสนอลูกค้า จนไฟล์มีขนาดใหญ่ สูตรซับซ้อน และข้อมูลมากขึ้นตามลำดับ

💡 แรงบันดาลใจในการต่อยอด

จุดเปลี่ยนเกิดขึ้นเมื่อได้พูดคุยกับพี่สาวที่น่ารักคนหนึ่ง ซึ่งมักให้คำแนะนำดีๆ อยู่เสมอ มีอยู่ครั้งหนึ่งเป็นเรื่องเกี่ยวกับการวางแผนการเงิน แม้ไม่รู้ว่าพี่เขาจำได้ไหมซึ่งเขาบอกว่าไฟล์ที่ผมใช้งานดูดีขึ้นมากเลย บางคนที่มี CFP ที่เคยพบยังทำไม่ได้เท่านี้ ถึงแม้ผมจะมีเพียงใบอนุญาตตัวแทนประกันและใบอนุญาตขายกองทุน (ทำเอาตัวลอยเลยทีเดียว) ซึ่งนั่นเองคือจุดเริ่มต้นของความตั้งใจที่จะทำโปรแกรมให้ทุกคนเข้าถึงได้ เพื่อใช้วางแผนการเงินส่วนบุคคลที่พัฒนาจากประสบการณ์การทำงานของตัวแทนประกันชีวิต และในที่สุดด้วยพลังของ AI แอพเวอร์ชัน 1 จึงได้เกิดขึ้น ถึงแม้ฟังก์ชั่นเพื่อการวางแผนการเงินจะยังไม่ครบทั้งหมด แต่เชื่อว่าจะทำให้ท่านได้เห็นภาพรวมการวางแผน เห็นความสำคัญของอัตราดอกเบั้ย และได้ตระหนักว่าเวลาเป็นสิ่งสำคัญ หากเริ่มช้าจะทำให้ต้องใช้เงินก้อนใหญ่ในการวางแผนระยะสั้นเพื่อเป้าหมายที่ต้องการ ซึ่งอาจจะทำให้ต้องเลื่อนการเกษียณออกไป

🎁 เจตนารมณ์ในการแบ่งปัน

ผมหวังว่าแอพนี้จะเป็นประโยชน์สำหรับผู้ที่ได้มาพบและทดลองใช้ ซึ่งผู้พัฒนาอนุญาตให้ท่านนำไปใช้สำหรับตนเองและครอบครัวได้อย่างอิสระ ไม่มีลิขสิทธิ์ใดๆ หรือถ้าจะอ้างถึงบ้างก็จะขอบคุณมากๆ ครับ

สำหรับคำแนะนำติชม หรือต้องการสอบถามเกี่ยวกับการวางแผน สินค้าประกันชีวิต สามารถติดต่อได้ตามข้อมูลด้านล่างนี้เลยครับ
